# BOT relatórios de mensagens WhatsApp

Este projeto visa a criação de um bot que ao ser adicionado em um grupo de whatsApp aguarda o recebimento de mensagens e procura por um arquivo .zip ou .txt. Ao receber, o bot ira baixar, processar e gerar relatórios com diversos gráficos com base nas mensagens trocadas, e enviar automaticamente no mesmo chat que foi recebido o arquivo.

## Rodar comando para instalação de dependências
```bash
pip install -r requirements.txt
```
## Tecnologias
- Linguagem: Python
- Framework: Node.js
